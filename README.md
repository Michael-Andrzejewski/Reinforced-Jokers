# Reinforced Jokers

By **Soareverix**.

> Work in progress.

A Balatro mod that keeps **Showman** permanently active and lets duplicate Jokers **stack** instead of taking a new slot. Pick up a Joker you already own and it combines with the one you have, so you can run a 3x Photograph or a 2x Chad. A stacked Joker triggers once per stack.

Built on commission for [Bean](https://www.youtube.com/@BeanBalatro).
